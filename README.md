# 🚖 Smart Taxi Rank Management System

A comprehensive taxi rank management system built for South African taxi associations. Digitizes queue management, tracks trips, manages drivers & vehicles, and provides real-time reporting.

## 👨‍💻 Developed by Khauhelo Sam

### ✨ Features
- **Digital Queue Management** - No more fighting over positions
- **Trip Tracking** - Complete trip history with earnings
- **Multi-User Roles** - Admins, Marshals, Owners, Drivers
- **Real-time Reports** - Revenue, passenger counts, route performance
- **Mobile Ready** - PWA support for drivers

### 🛠️ Tech Stack
- PHP 7.4/8.0
- MySQL
- Bootstrap 5
- JavaScript
